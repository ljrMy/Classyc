这是一个简洁、方便的课程工具软件

## 功能介绍
![界面](https://github.com/ljrMy/Classyc/blob/main/static/layer.jpg?raw=true)
1. 查看星期与天气
2. 查看课表（双击打开对应路径）
3. 课堂小工具
4. 倒数日
5. 可以隐藏起上边栏
